# 🌟 星绘 Celestia 网站部署教程

> **目标**：将星绘角色介绍网站部署到你的阿里云服务器，绑定域名 `celestia.codes`
>
> **难度**：⭐ 零基础友好，一步一步跟着做就行

---

## 📋 准备工作

在开始之前，请确认你已经拥有：

| 项目 | 你的信息 |
|------|---------|
| 域名 | `celestia.codes` |
| 服务器IP | `39.101.73.36` |
| 服务器系统 | 宝塔面板已安装 |

---

## 第一步：下载网站文件

1. 从本对话下载 `celestia-site.zip` 文件
2. 解压后你会得到一个 `celestia-site` 文件夹，里面有：
   ```
   celestia-site/
   ├── index.html      ← 网页主文件
   └── images/         ← 所有图片
       ├── img01.png
       ├── img02.png
       └── ...
   ```

---

## 第二步：登录宝塔面板

1. 打开浏览器，访问：`http://39.101.73.36:8888`
   - 如果端口不是8888，请使用你实际安装的宝塔端口
2. 输入宝塔面板的用户名和密码登录

---

## 第三步：创建网站

### 3.1 添加站点

1. 在宝塔面板左侧菜单，点击 **「网站」**
2. 点击右上角的 **「添加站点」** 按钮
3. 在弹出的窗口中填写：

   | 选项 | 填写内容 |
   |------|---------|
   | 域名 | `celestia.codes` |
   | 根目录 | `/www/wwwroot/celestia.codes`（保持默认即可） |
   | FTP | 不创建 |
   | 数据库 | 不创建 |
   | PHP版本 | 纯静态（不需要PHP） |

4. 点击 **「提交」**

### 3.2 上传网站文件

1. 在网站列表中找到 `celestia.codes`，点击 **「根目录」**（或目录路径）
2. 进入后会看到默认生成的文件（如 `index.html`、`404.html` 等）
3. **全选所有文件 → 删除**（清空目录）
4. 点击 **「上传」** 按钮
5. 选择 **「上传目录」**，然后选择你解压好的 `celestia-site` 文件夹
6. 等待上传完成（约2-5分钟，取决于网速）

> ⚠️ **重要**：上传完成后，确保目录结构是这样的：
> ```
> /www/wwwroot/celestia.codes/
> ├── index.html
> └── images/
>     ├── img01.png
>     └── ...
> ```
> 不要有多余的 `celestia-site` 这层文件夹！

---

## 第四步：域名解析（关键步骤！）

### 4.1 登录域名控制台

1. 打开 [阿里云域名控制台](https://dc.console.aliyun.com/)
2. 登录你的阿里云账号
3. 找到你的域名 `celestia.codes`，点击 **「解析」**

### 4.2 添加解析记录

点击 **「添加记录」**，填写以下信息：

| 记录类型 | 主机记录 | 解析线路 | 记录值 | TTL |
|---------|---------|---------|--------|-----|
| A | @ | 默认 | `39.101.73.36` | 10分钟 |
| A | www | 默认 | `39.101.73.36` | 10分钟 |

解释：
- `@` 表示主域名 `celestia.codes`
- `www` 表示 `www.celestia.codes`
- 记录值填写你的服务器IP

点击 **「确认」** 保存。

> ⏰ 解析生效需要 **10分钟~2小时**，请耐心等待。

---

## 第五步：配置SSL证书（HTTPS）

为了让网站可以通过 `https://celestia.codes` 访问（更安全，浏览器不会显示"不安全"），需要配置SSL证书。

### 5.1 申请免费证书

1. 回到宝塔面板，点击左侧 **「网站」**
2. 找到 `celestia.codes`，点击 **「设置」**
3. 在弹出的窗口中，点击左侧 **「SSL」**
4. 选择 **「Let's Encrypt」** 标签
5. 勾选域名 `celestia.codes` 和 `www.celestia.codes`
6. 点击 **「申请」**
7. 等待申请完成（约1-2分钟）

### 5.2 强制HTTPS

1. 证书申请成功后，开启 **「强制HTTPS」** 开关
2. 点击 **「保存」**

---

## 第六步：访问你的网站！

等待域名解析生效后，在浏览器中访问：

```
https://celestia.codes
```

🎉 如果看到星绘的精美网页，说明部署成功！

## 🔧 常见问题排查

### Q1: 访问域名显示 "无法访问此网站"

**可能原因**：域名解析还没生效
- 等待 10分钟~2小时
- 或者尝试访问 `http://39.101.73.36` 看能否打开（如果能，说明网站没问题，只是解析没生效）

### Q2: 访问IP能打开，但域名打不开

**可能原因**：域名解析配置错误
- 检查阿里云解析记录是否正确
- 确认记录值是 `39.101.73.36`
- 使用 [DNS查询工具](https://tool.chinaz.com/dns/) 检查解析是否生效

### Q3: 网页样式错乱或图片不显示

**可能原因**：文件路径不对
- 检查宝塔中文件路径是否为 `/www/wwwroot/celestia.codes/index.html`
- 确认 `images` 文件夹和 `index.html` 在同一目录下

### Q4: 宝塔面板打不开

**可能原因**：端口未开放
- 登录阿里云ECS控制台
- 找到你的服务器，进入 **「安全组」**
- 添加规则，开放端口：`8888`（宝塔）、`80`（HTTP）、`443`（HTTPS）

### Q5: 如何更新网站内容？

- 修改 `index.html` 文件：在宝塔面板的「文件」中找到并编辑
- 替换图片：在宝塔「文件」中进入 `images` 目录，上传新图片覆盖旧图片

## 🚀 进阶玩法（可选）

### 添加网站统计

1. 注册 [百度统计](https://tongji.baidu.com/) 或 [Google Analytics](https://analytics.google.com/)
2. 获取统计代码
3. 在宝塔面板中编辑 `index.html`
4. 将统计代码粘贴到 `</body>` 标签之前

### 添加评论功能

可以使用 [Disqus](https://disqus.com/) 或 [Valine](https://valine.js.org/) 添加评论系统。

### 添加背景音乐

在 `index.html` 的 `<body>` 中添加：
```html
<audio autoplay loop>
    <source src="bgm.mp3" type="audio/mpeg">
</audio>
```
